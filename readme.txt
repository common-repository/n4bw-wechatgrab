N4BW-WechatGrab WordPress 微信采集助手

== Description ==

N4BW-WechatGrab 是专门为采集微信公众号文章所提供的 WordPress 插件.
包括采集(文章\公众号所有文章\文章阅读数\文章点赞数\文章评论数\文章评论等),提供丰富的设置,让你链接公众号更加便捷.

= 1.1.3 =
* 添加视频自适应设置(支持视频自适应比例)
* 添加文章地址过期提示
* 解决视频无法采集问题
* 解决部分文章图片无法采集显示防盗链问题
* 解决部分情况下图片加载出现错误的情况
* 修改样式初始化加载位置至文档 wp_head 处
* 修复大量 BUG

使用文档： https://nnnn.blog/n4bw-wechatgrab.html

== Installation ==

1. 上传到 `/wp-content/plugins/` 目录
2. 在后台插件菜单激活该插件
3. 请确保您的 PHP 程序版本至少为 5.3.0 或以上
